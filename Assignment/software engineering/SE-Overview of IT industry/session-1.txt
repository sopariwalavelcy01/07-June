Q-1 : Name 5 IT job roles and write one line about what each role does.
ANS : Here are 5 common IT job roles and what they do:
1.Software Developer – Designs, builds, and maintains software applications and systems.
2.System Administrator – Manages and supports an organization's computer systems, servers, and networks.
3.Network Engineer – Designs, implements, and maintains network infrastructure to ensure reliable connectivity.
4.Cybersecurity Analyst – Protects computer systems and data by monitoring for threats and implementing security measures.
5.Database Administrator (DBA) – Manages databases, ensuring data is stored securely, efficiently, and remains accessible to users.

Q-3 : Write the full forms and one-line use of: GitHub, Slack, Jira, IDE, API.
ANS : 
1.GitHub
Full Form: Git-based Hosting Platform

2.Slack
Full Form: Searchable Log of All Communication and Knowledge
Use: Used for team communication and collaboration.

3.Jira
Full Form: Jira is not an acronym; it is a product name.
Use: Used to track tasks, bugs, and project progress.

4.IDE
Full Form: Integrated Development Environment
Use: Used to write, test, and debug software code.

5.API
Full Form: Application Programming Interface
Use: Used to enable communication between different software applications.

Q-5 : What is the difference between a developer and a tester? Answer in 2 lines.
ANS : A developer writes, modifies, and maintains software code to build applications and features.

A tester checks the software for bugs, errors, and quality issues to ensure it works correctly before release.
