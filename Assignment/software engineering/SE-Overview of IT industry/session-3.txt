Q-1 : Write the full form and one-line use of: HTTP, HTTPS, FTP, SMT
ANS : 
1.HTTP
Full Form : HyperText Transfer Protocol
Use : Used for transferring web pages and data between a browser and a web server.

2.HTTPS
Full Form : HyperText Transfer Protocol Secure
Use : Used for securely transferring data between a browser and a web server using encryption.

3.FTP 
Full Form : File Transfer Protocol
Use : Used for transferring files between computers over a network.

4.SMTP
Full Form : Simple Mail Transfer Protocol
Use : Used for sending emails from one mail server to another.


Q-2 : What is the difference between IPv4 and IPv6? Answer in 2 lines.
ANS :
IPv4 uses 32-bit addresses and supports about 4.3 billion unique IP addresses (e.g., 192.168.1.1).
IPv6 uses 128-bit addresses and supports a vastly larger number of unique IP addresses (e.g., 2001:db8::1).

Q-4 : What do these HTTP status codes mean: 200, 301, 404, 500, 403?
ANS :
200 : The request was successful, and the server returned the requested resource.
301 : The requested page has been permanently moved to a new URL.
404 : The server could not find the requested page or resource.
500 : The server encountered an unexpected error while processing the request.
403 : The server understood the request but denied access to the resource.


Q-5 Explain DNS hierarchy (root, TLD, domain, subdomain) with www.google.com as an example.
ANS :
Using [www.google.com](http://www.google.com) as an example, the DNS hierarchy works as follows:

1. Root Domain (.)
The root is the highest level in the DNS hierarchy and is represented by a dot (**.**). It directs queries to the appropriate Top-Level Domain (TLD) servers.

2. TLD (Top-Level Domain) – .com
The .com part is the TLD. It tells DNS which domain registry is responsible for the domain name.

3. Domain Name – google.com
The google part is the domain name registered under the .com TLD. It identifies Google's website on the internet.

4. Subdomain – [www.google.com](http://www.google.com)
The www part is a subdomain of google.com. It is commonly used to identify the web service of the domain.

Hierarchy:
Root (.) → .com (TLD) → google.com (Domain) → [www.google.com](http://www.google.com) (Subdomain)**.
