Q-2 : List three popular programming languages you have heard of, and for each, mention whether it is usually compiled, interpreted, or can be both.<br><br><em><strong>Hint:</strong> Think of languages like Java, Python, and JavaScript.</em>
ANS : Three Popular Programming Languages

1. Java
   * Type : Compiled and Interpreted (Both)
   * Java source code is compiled into bytecode, which is then interpreted by the Java Virtual Machine (JVM).

2. Python
   * Type : Interpreted
   * Python programs are executed by the Python interpreter.

3. JavaScript
   * Type : Interpreted
   * JavaScript code is executed by the web browser's JavaScript engine.

Q-4 :  Write down the steps (as an algorithm, using simple numbered instructions) for ordering food on Swiggy — from opening the app to completing payment.
ANS :
 Algorithm: Ordering Food on Swiggy

1. Start.
2. Open the Swiggy app.
3. Log in to your account.
4. Select or confirm your delivery location.
5. Search for the restaurant or food item.
6. Choose the food items you want to order.
7. Add the selected items to the cart.
8. Review your order and proceed to checkout.
9. Select your preferred payment method.
10. Complete the payment.
11. Receive the order confirmation.
12. Stop.
