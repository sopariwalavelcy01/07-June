Q-2 : Open VS Code or DevC++ and create a new file named hello.c. Write a C program that prints 'Hello, Instagram World!' to the console.
ANS :
#include <stdio.h>

int main() {
    printf("Hello, Instagram World!");
    return 0;
}

OUTPUT : Hello, Instagram World!


Q-3 : Identify and label the header, main function, and return statement in your hello.c program using comments.
ANS :  #include <stdio.h>   // Header file for input and output functions

int main()           // Main function: Program execution starts here
{
    printf("Hello, Instagram World!");

    return 0;        // Return statement: Ends the program successfully
}


Q-4 : Compile your hello.c file using the GCC command line (gcc hello.c -o hello) and run the output file to display the message in your terminal.
<br><br><em><strong>Hint:</strong> On Windows, the output file will be hello.exe; on Linux/Mac, it will be hello.</em>
ANS :
RUN : 
    gcc hello.c -o hello
    hello

OUTPUT : Hello, Instagram World!

If you are using Linux/macOS, run:
    gcc hello.c -o hello
    ./hello

OUTPUT : Hello, Instagram World!
